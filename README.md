## Overview
- Adds unique first-person animations to key story moments in New Vegas and Tale of Two Wastelands.

- [xNVSE](https://www.nexusmods.com/newvegas/mods/67883)
- [JIP LN NVSE](https://www.nexusmods.com/newvegas/mods/58277)
- [JohnnyGuitar NVSE](https://www.nexusmods.com/newvegas/mods/66927)
- [kNVSE](https://www.nexusmods.com/newvegas/mods/71336)
- [lStewieAl's Tweaks](https://www.nexusmods.com/newvegas/mods/66347)

## Installation
Installs like any other mod, manually or using a mod manager.
